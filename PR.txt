JSON Treatment
O ficheiro maps.json foi dividido manualmente em 2 outros ficheiros json, um com as cidades outro com as ligaçoes entre estas.
Estes dois json files foram importados para a mongoDB utilizando o comando mongoimport, para collections diferentes como no seguinte exemplo:
    mongoimport --db MAPA2022 --collection cidades --file cidades.json --jsonArray
    mongoimport --db MAPA2022 --collection ligacaos --file ligacoes.json --jsonArray
    
Foi feito um simples Schema para ambas as collections na secção dos models e o respetivo controller na pasta Controllers.

Executar a aplicação - npm start
